GitHub

Listing and Adding Remote Repositories

git remote -v : List all remote repositories

git remote add origin <repo url> : Add an existing local repository to GitHub 

\\remote - URL for hosted repo
\\origin - default remote name - we can define another name as well

Example:

git remote add novelrepo <repo url>

Pushing to Remote Repositories

git push <remote> <localbranch>:<remotebranch> : Create the remote branch (if not existing) on GitHub and push the local branch

Example:

git push origin bug_fix:master

git push -u <remote> <localbranch> : Push the local branch to the remote repository and set the upstream. After this, you can simply use git push to push changes to the branch.
